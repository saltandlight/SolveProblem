# 힙의 개념

## 1. 힙의 개념

**부모의 값이 항상 자식보다 작은 `완전 트리`**

![](./cap1.JPG)

- 부모일수록 우선순위가 높다. 
- 완전 이진 트리: 왼쪽에서 오른쪽으로 노드들이 채워짐
- 1) 값을 추가할 수 있다.
- 2) 값을 제거할 수 있다.
- 힙의 루트가 가장 우선순위가 높다. 
- 우선 순위 높은 것 바로 알 수 있고 바로 제거 가능 (우선순위 큐의 기능)
- 그런데 값 추가, 제거 시 힙을 재구성해야 함
